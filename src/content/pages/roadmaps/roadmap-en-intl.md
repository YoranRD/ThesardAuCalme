This roadmap focuses on first-month execution for international PhD candidates in France.

## Milestones

- Core admin tracker created and active.
- Enrollment process initiated with proof archive.
- Residence-related steps mapped by deadline.
- Health onboarding started.
- Weekly research routine protected (minimum three sessions).

## Common pitfalls

- Doing only admin work for multiple weeks.
- Losing document traceability.
- Waiting for "perfect conditions" to start writing.
- Using too many apps without one source of truth.

## 30-day execution rhythm

### Week 1

- Access setup + admin map + first supervisor contact.

### Week 2

- Enrollment and residence file progression.

### Week 3

- Health steps + stable writing/reading blocks.

### Week 4

- Gap audit + month-two planning.

## Minimal review template

- What was completed?
- What is blocked?
- What is the next single priority?

A short weekly review keeps the roadmap operational.
